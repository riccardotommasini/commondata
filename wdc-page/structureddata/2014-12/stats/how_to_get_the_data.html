Download Instructions for the WDC RDFa, Microdata, and Microformats Data Sets (December 2014)

This document contains instructions on how to download the December 2014 version of the Web Data Commons RDFa, Microdata, and Microformats data sets.

Download the Extracted Data

The extracted RDFa, Microdata and Microformat data is provided for download as N-Quads. Files are compressed using GZIP and each file is around 100 MB large. Overall 3,581 files with a total size of 373 GB are provided.

List of download URLs for RDF from the Winter 2014 corpus (Example Content)

The extracted RDF data can be downloaded using wget with the command wget -i http://webdatacommons.org/structureddata/2014-12/files/file.list. Get the Code

The source code can be checked out from our Subversion repository. The extraction of Winter 2014 was done with version 1.0.1 of the extractor. For more information about the framework and a detailed description how to run a own extraction visit the framework page.
